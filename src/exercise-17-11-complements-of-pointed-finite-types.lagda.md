## Problem statement

For any `k : ℕ`, construct an equivalence

```text
  (Σ(X : BS_{k + 1}) X) ≃ BS_k.
```

In other words, show that the type of `(k + 1)`-element types equipped with a
point is equivalent to the type of `k`-element types.
Conclude that the type of pointed finite types is equivalent to the type of
finite types, i.e., conclude that we have an equivalence

```text
  (Σ(X : 𝔽) X) ≃ 𝔽.
```
